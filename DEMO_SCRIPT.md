# Demo Walkthrough Script

## 1) Intro (15s)

- Open Photoshop and show Color Blind Guide panel.
- Briefly explain: simulate, scan, correct, export.

## 2) Simulation (30s)

- Select `Simulate` mode and choose `Deuteranopia`.
- Adjust intensity.
- Click `Apply Fix`.
- Show created adjustment layer and visual result.

## 3) Scan and WCAG (40s)

- Open layered design with editable colors.
- Click `Scan Document`.
- Explain conflict pairs, contrast ratio, and WCAG status.

## 4) Suggestions (30s)

- Show suggestions list.
- Select replacements and click `Apply selected`.
- Re-scan to show improvement.

## 5) Flattened Artwork Correction (25s)

- Open single-layer image.
- Set mode to `Correct`.
- Click `Apply Fix`.
- Show global correction layer fallback.

## 6) Export (20s)

- Export TXT report.
- Export PDF report.

## 7) Revert and Close (15s)

- Click `Revert Last Preview`.
- Summarize value: accessibility checks directly in design workflow.
