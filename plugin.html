Install in 4 steps.

Adobe Photoshop UXP Plugin

  1. 01

    Download VisionX Plugin

    Click the Download button above to get the VisionX-Photoshop-Plugin.zip file. Save it somewhere easy to find like your Desktop.

    Download .zip →
  2. 02

    Open Adobe UXP Developer Tool

    Launch Adobe Photoshop. Go to Plugins menu → Development → UXP Developer Tool. If you don't see this, go to Edit → Preferences → Plugins and enable Developer Mode first.

    Enable Developer Mode in Preferences → Plugins
  3. 03

    Add the Plugin

    In the UXP Developer Tool window, click Add Plugin. Navigate to the unzipped VisionX-Photoshop-Plugin folder and select the manifest.json file. Click Load Selected.

  4. 04

    Open VisionX Panel

    Go back to Photoshop. Open Plugins menu → VisionX → Open Panel. The VisionX panel will appear. Open any image and select a simulation mode to begin.

    You're ready!
